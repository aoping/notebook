## 集中式工作流
- git init --bare /path/to/repo.git
- git clone
- git status # 查看本地仓库的修改状态
- git add # 暂存文件
- git commit # 提交文件
- git push (git push origin master)
```
主流的git版本，可以省略后面2个参数：远程仓库别名、推送分支
```
- git pull --rebase (git pull --rebase origin master)
- git add <some-file> 
- git rebase --continue
- git rebase --abort
